# AEE2004 Low Speed (CONF+CAR) buses

## Common frames

| ID range       | Description         |
|----------------|---------------------|
| 0x480 -> 0x4BF | Error event frames  |
| 0x400 -> 0x43F | Wakeup event frames |
| 0x500 -> 0x53F | Supervision frames  |
| 0x5C0 -> 0x5EF | Version frames      |
